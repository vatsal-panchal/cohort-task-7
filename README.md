# DOM Explorer — Task Manager

DOM Explorer is a responsive three-column dashboard built using HTML, CSS, and JavaScript. The project combines task management with demonstrations of important DOM concepts such as attributes vs properties, event propagation, and the browser rendering pipeline.

## Live Demo

[https://cohort-task-7.vercel.app/](https://cohort-task-7.vercel.app/)

---

## Overview

The application is divided into three sections:

### Left Sidebar

* Branding section
* Navigation menu
* Theme switcher

### Main Workspace

* Task statistics
* Task creation form
* Search and filter controls
* Task list

### Right Inspector Panel

* Displays information about the selected task
* Shows attributes and dataset values
* Provides a live view of the DOM node

---

## Features

### Task Management

* Add tasks
* Edit tasks
* Delete tasks
* Mark tasks as completed
* Clear all tasks
* Store tasks using Local Storage

### Search and Filtering

* Search tasks by title
* Filter tasks by category
* Filter tasks by status

### Statistics Dashboard

* Total tasks
* Pending tasks
* Completed tasks
* Number of categories

### Theme Support

* Light mode
* Dark mode
* Theme preference persistence using Local Storage

## DOM Concepts Demonstrated

### Attributes vs Properties

Demonstrates:

* `getAttribute()`
* `setAttribute()`
* `removeAttribute()`
* `hasAttribute()`
* `dataset`

### Event Propagation

Illustrates:

* Event Bubbling
* Event Capturing

### Browser Rendering Pipeline

Shows how the browser processes:

1. HTML
2. Parsing
3. Tokenization
4. DOM Tree
5. CSS
6. CSSOM Tree
7. Render Tree
8. Layout
9. Paint
