# Cohort Task 7

A clean and simple three-column dashboard layout built using HTML and CSS. It features a modern design that opens in dark mode by default and adapts easily to mobile screens.

Live Link: [https://cohort-task-7.vercel.app/](https://cohort-task-7.vercel.app/)

## Layout Sections

The webpage is divided into three main parts:

* Left Sidebar: Contains the branding logo, navigation links, and the light/dark theme switch.
* Central Area: The main space used for displaying stats, filling out forms, and viewing the task list.
* Right Panel: A dedicated section to inspect and view the specific details of a selected item.

## Key Features

* Dark theme is enabled by default for a comfortable viewing experience.
* Designed with a clean look, using soft backgrounds instead of heavy borders.
* Responsive design that automatically adjusts and collapses neatly on smaller smartphone screens.
